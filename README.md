# Evaluación de la Seguridad Vial y Siniestralidad en Latacunga

## 📌 Descripción del Proyecto
Este repositorio contiene el análisis de política pública sobre la seguridad vial en Latacunga (Cotopaxi). Evalúa la efectividad de la LORTTSU y el control por fotorradares sobre los índices de siniestralidad vial a partir de los datos del INEC y la ANT.

## 🤖 Arquitectura Multiagente
El proyecto fue construido utilizando una arquitectura multiagente coordinada en VS Code / Kiro:
* **Agente de Programación y Datos:** Procesamiento y limpieza de microdatos del INEC (69 siniestros).
* **Agente de Auditoría Metodológica:** Revisión de consistencia y reproducibilidad de indicadores.
* **Agente de Política Pública:** Formulación de recomendaciones estratégicas y viabilidad.
